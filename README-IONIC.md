## Build
`nx build corny-notes-ng`

### Web
`npm i`

`nx run corny-notes-ng:add:android`
`nx run corny-notes-ng:build`

### Android
`npm i`

Only first time:

`nx run corny-notes-ng:add:android`

Build and sync:

`nx run corny-notes-ng:build`

`nx run corny-notes-ng:sync:android`

To run in android-studio:

`nx run corny-notes-ng:open:android`
